## Prezentace – Solid storage drives
Hodnocení: full

Během distanční výuky jsme prezentaci nahrávali a vytvářeli k ní transkript – slidy i transkript zde tedy zveřejňuji, ale prosím laskavého čtenáře, aby oželel nahrávku.
